\projectheading{DMARC Report Dashboard}{TypeScript, AWS Lambda}{2026}
\begin{itemize}[topsep=2pt]
    \item Event-driven AWS application that processes DMARC reports received by email
    \item Scheduled Lambda function polls an inbox hourly, extracting report attachments and storing them in S3; a second Lambda triggered on S3 PUT events parses each report and writes results to DynamoDB
    \item React (TypeScript) frontend provides a dashboard with graphs displaying DMARC pass/fail rates across domains over time
\end{itemize}
